# SpoonFeed Recipe Plugin

## Description

Add thousands of different recipes to your WordPress site from https://spoonacular.com .
This plugin is set to query https://spoonacular.com recipes, with the ability to query results based on ingredients.
Improve SEO by adding recipes to your blog. 
See screenshots folder

## Instruction

- Visit https://spoonacular.com for API key
- Set Values in Admin Dashboard

## Features
- ShortCodes
- Easily change from grid and Table view
- Singular recipe view with related recipes
- Search
